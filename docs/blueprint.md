# **App Name**: Contract Catalyst

## Core Features:

- Contract Value Input: Input field for the contract value.
- Renewal Year Input: Input field for the contract renewal year.
- Issue Year Input: Input field for the contract issue year.
- Contract Calculation: Back and logic in backend ( contract value/(( renewal year - issue year)*12)= Value1, Value1*10.33= value2 Ans= (value2 /100)+Value1

## Style Guidelines:

- Primary color: A soft, professional blue (#5DADE2) to inspire confidence and reliability.
- Background color: A very light blue (#EBF4FA), almost white, providing a clean backdrop.
- Accent color: A complementary orange (#F39C12) for key actions and highlights, drawing the user's attention without overwhelming the interface.
- Font: 'Inter', a sans-serif font, used for both headlines and body text, because its clean, neutral design is ideal for creating a professional impression.
- A clean and structured layout to make input fields easily accessible. Input fields are vertically stacked for clear presentation.
- Simple and recognizable icons for each field label.
